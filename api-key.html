getting an API key

Now that you have your website, here's how to actually call the API. In order to do that, we're going to need an API key.

If you want to learn more about what all of this means, check out this page.

First, head over to thecatapi.com, and click "GET YOUR API KEY".

It will then prompt you to choose between a free and a paid plan. You'll only need a free account, so select that option.

Then, you'll have to create an account. Enter an email and write a short app name (mine was CatAPI tutorial)

After that, click submit and you'll be ready to go! You'll know that you're set when you see the message below:

Once you see this message, click the button below to learn how to call the API in your website!

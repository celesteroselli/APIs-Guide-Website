Getting started with CatAPI

In order to get started, we're going to need somewhere to call the API from.

If you already have an application you've created in mind, then that's great! If not, you can follow the guide here to make a simple website.

Please keep in mind that if you want to follow the 2nd part of this guide which covers using API keys, you'll need to be using a Javascript framework. This could be Astro, React, Node.js, or anything else. This guide will cover Astro, so if you want to follow along for using an API key you can start with a simple astro template from this guide.

Since we'll be using CatAPI to get an image, make sure whichever project you are planning on using this for has an <img> tag in the html too (the guide covers this in case your project doesn't have one yet)!
